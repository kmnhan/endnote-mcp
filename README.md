# MCP Adapter for EndNote (Unofficial)

Unofficial read-only MCP adapter for user-configured local EndNote libraries.

The Codex plugin bundle lives in
[`plugins/endnote-local-adapter`](plugins/endnote-local-adapter/). Its
plugin-local README is the single source of truth so the plugin folder remains
self-contained when installed, copied, or packaged.
